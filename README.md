# analytics-portfolio
Portfolio Optimization & Factor Analysis
**Objective**

Constructed and evaluated a diversified 10-stock portfolio from the S&P MidCap 400 using mean-variance optimization and multi-factor regression analysis.

**Tools Used**:

Python (Pandas, NumPy, SciPy), Statsmodels, Matplotlib

**Methodology**

Calculated annualized returns and volatility
Built correlation matrix
Performed mean-variance optimization to determine optimal asset weights
Conducted out-of-sample backtesting
Ran Fama-French 5-factor regression to analyze systematic risk exposure

**Key Results**

Optimized portfolio achieved annualized return of X%
Market beta of 0.80 indicates moderate systematic risk
Significant exposure to SMB and CMA factors
No statistically significant alpha

**Business Insight**

The portfolio demonstrates diversified sector exposure with moderate market sensitivity and identifiable factor tilts, illustrating practical application of quantitative portfolio construction and risk modeling.
